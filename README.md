# 💳 Credit Card Fraud Detection System

A full-stack machine learning project that detects fraudulent credit card transactions using Logistic Regression.

## 🚀 Features
- Predicts whether a transaction is fraud or not
- Machine Learning model (Logistic Regression)
- Interactive frontend using React
- API-based backend using Python (Flask/FastAPI)
- Visualization and comparison of results

## 🛠️ Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Python
- ML Model: Logistic Regression (Scikit-learn)
- API: Flask / FastAPI
- Deployment: Vercel (Frontend), Render (Backend)
